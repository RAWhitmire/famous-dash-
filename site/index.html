How to use this site

This is reference + schedule

No live KPI fetches, no dashboards that update automatically — those wasted credits and weren't accurate anyway. Anything date-driven gets edited by hand when the plan actually changes.

Filming material is offline

Sprint scripts live as markdown + PDF in the project folder, ready for iPad. The Filming Materials Index lists them all.

Memory + voice rules persist

Voice, audience, pillars, and scrub rules live in the Foundation + Production Playbook. Read those before writing anything new.

Update when it actually shifts

Sprint dates, hero markers, key inputs — edit the relevant page when something material changes. No need to keep it live-synced.
