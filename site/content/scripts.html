Pre-written hooks + script structures for the formats that actually win. Reference + remix · don't read scripts on camera, internalize them then perform.

How to use scripts

Script the hook tightly. Improvise the body. Script the CTA tightly. The first 1.5 seconds and the last 3 seconds are the only places where exact wording matters. The middle is performance.

Reading from a teleprompter for the whole video reads as wooden. The audience can clock it.

Script structure (universal)
  1. Hook (1.5 sec): tight, scripted, hits the curiosity gap
  2. Setup (3–5 sec): orient the viewer — who/what/why
  3. Body (15–40 sec): the meat. Improvise but stay on point.
  4. Turn (3–5 sec): the twist, the reveal, the punchline
  5. CTA (2 sec): tight, scripted. "Save this · share with the friend who [thing] · part 2 if this hits"
  6. Loop or hard cut: abrupt cut for memorable. Loop back to hook for rewatch.
