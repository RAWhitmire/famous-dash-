Filming Day Shot Lists

2 batch days/week · 4 light days · 2 zero-film days. Each has its own shape — bring the right energy.

★ BATCH DAY (Wed)

Wednesday 9 AM – 4 PM. 10–14 main + 4 DT in one shoot.

  1. Block A (9–12): warm-ups + 4 medium-effort (Direct Address, Storytime)
  2. Block B (1–4): 4–6 high-effort (DT character, Sketch, Process)
  3. Block C (3:30–4): 2–4 quick-fire (Carousels, voiceovers, B-roll)

Friday-night → Sunday-evening = 4 days off filming if Wed lands.

★ BATCH DAY (Sun, optional)

Sunday 3:30 PM – 5 PM (after acting practice). Light filming only — voiceovers + B-roll bank.

  • Voiceovers for B-roll already shot
  • Direct-address takes (3–5 in 30 min)
  • Photo dump for carousels
  • Reaction/duet content (low-prep)

☆ LIGHT DAY (Mon, Thu)

Print shop day eves. 30–60 min after dinner. Low-prep content only.

  • Reply-to-comment (Q&A)
  • Direct address rambles
  • Trend remix (if a sound is hot)
  • Carousel content

✗ NO FILM DAY (Fri, Sat)

Pure post-from-queue days. If you film, it's voluntary.

  • All 3 slots ship from the queue you built Wednesday
  • If queue is empty by Friday → emergency Sunday batch + check Get Ahead
  • Friday/Saturday treat night allowed
Shot list essentials (every batch day)

Before you start filming:

  • ✅ Camera battery 100% (charge night before)
  • ✅ Memory card cleared / backed up
  • ✅ Tripod set + ring light positioned
  • ✅ Lavalier mic checked or boom mic placed
  • ✅ Cardboard set / DT props staged (if DT day)
  • ✅ Costume ready (uniform, glasses, anything)
  • ✅ Hook list visible (write on a sticky note, post in eyeline)
  • ✅ Water bottle full, snacks within reach
During the shoot

Reduce decisions

  • One outfit per block (don't re-costume between every video)
  • Same lighting setup all day
  • Same camera angle for talking-head batches
  • Switch only when format changes

Capture extras

  • Always 2 takes minimum — best take is rarely the first
  • 30 sec of "neutral" B-roll between videos (apartment, hands, cat)
  • Bloopers naturally — these are content too
  • Reset shots (clean room, clean desk) every hour for editor
