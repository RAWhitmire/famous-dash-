Card structure (what each post specifies)

Always defined

  • ID — M01–M93 (main) or DT-01–DT-32
  • Day + slot — when it ships
  • Title — working name
  • Topic — what the post is about
  • Verbal hook — first 1.5 sec script
  • Visual hook (frame 1) — what we SEE first
  • CTA — what we ask the viewer to do
  • Ending mechanic — loop / cliffhanger / abrupt cut
