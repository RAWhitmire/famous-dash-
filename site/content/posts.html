Card structure — what every post specifies

Always defined per script

  • ID — M01–M93 (main) or DT-01–DT-25
  • Day + slot — when it ships
  • Title — working name
  • Format — F01–F15 (tags to Format Library)
  • Pillar — P1–P5 (per Foundation)
  • 4-stack hook per beat — VO / Visual / On-screen text / Audio cue or Pattern interrupt
  • 6-beat structure — Hook · Rehook 1 · Rehook 2 · Rehook 3 · Climax · Loop
  • CTA — CTA01–05, varied per video
  • Caption — for the post
  • First comment — pinned, often with PASSENGER lead-magnet line
  • Hashtags — 3 pillar-coded + 2 evergreen

Tracked after posting

  • Views at 24h / 7d
  • Completion % (target: 70% = ~31s on a 45s vid)
  • Comments / saves / shares
  • Followers gained from this post
  • Format Score — calculated for the format-test methodology
  • Live tracking happens in the Analytics page
