Weekly Rhythm

The standing pattern. Print shop Mon/Tue/Thu/Fri 10–6 + Sat 9–5. Wed + Sun OFF. 1hr commute each way + 20 min ready.

Mon · Tue · Thu · Fri PRINT SHOP 10–6

  • 7:00 AM wake · 7:00–7:20 ready · slot 1 queued post fires
  • 7:20–7:30 quick breakfast (Tier 1)
  • 8:40 AM leave · 9:40 arrive · 10–6 work
  • ~1 PM lunch break at print shop — slot 2 ships, eat (Tier 1)
  • 6 PM leave · 7 PM home
  • 7:00–7:30 dinner (Tier 1 / 3)
  • 7:30–10:00 edit / community / DT bandwidth · slot 3 ships
  • 10–11 wind down · 11 PM sleep target (8 hr in bed)

Tuesday REST EVENING

  • Same work timeline as other shop days
  • Eve: 🛌 NO EDIT NIGHT · slot 3 ships from queue (pre-edited Mon) · early sleep 10:30 PM
  • Recovery valve. Skip it and the wheels come off by week 3.
If Tuesday rest falls 2 weeks in a row, pause filming the next day, queue-only.

Wed OFF ★ PRIMARY BATCH

  • 8:00 AM wake (sleep in 1 hr) · big breakfast (Tier 1)
  • 9:00–12:00 film Block A warm-up + Block B peak
  • 12:00–1:00 real lunch break — proper food, not snacks
  • 1:00–4:00 Block B finish + Block C cool-down + DT block
  • 4:00–6:00 review takes · start edits · slot 2 ships
  • 6:00–7:00 dinner
  • 7:00–9:30 edit · queue tomorrow · slot 3 ships · DT bandwidth
  • 10:30 PM sleep · earlier than weekday — recover before 4-day work block
THE day. 10–14 main + 4 DT in one shoot. Friday-night → Sunday-evening = 4 days off filming.

Saturday PRINT SHOP 9–5

  • 7:00 AM wake (1 hr earlier than weekdays) · 7:20 ready
  • 7:40 AM leave · 8:40 arrive · 9–5 work
  • ~12:30 lunch break — slot 2 ships
  • 5 PM leave · 6 PM home
  • 6–7 dinner · 7–9 light edit + slot 3 from queue · 🎉 Friday/Sat treat night allowed
  • 10–11 wind down · 11 PM sleep

Sun OFF RECOVERY · OPTIONAL BATCH-COOK

  • 8:00 AM wake · slow morning · slot 1 ships from queue
  • 8:00–9:00 breakfast · start laundry load 1 · email/comment sweep
  • 9:00–10:30 groceries OR delivery (Tier 1 staples)
  • 10:30–12:30 🍱 OPTIONAL Tier 2 batch-cook — only if you feel like it
  • 12:30–1:30 lunch + slot 2 ships
  • 1:30–3:30 rotating chore (chore wheel) + laundry continues
  • 3:30–5:00 light filming if needed · voiceovers · B-roll bank
  • 5:00–6:00 15-min acting practice restart
  • 6:00–7:00 dinner
  • 7:00–9:00 queue Mon-Wed posts · DT bandwidth block
  • 9:00–10:00 slot 3 ships · wind down · 11 PM sleep before 4-day work block
Sunday is recovery + life maintenance. Don't let filming creep here unless it's voiceover-only.
